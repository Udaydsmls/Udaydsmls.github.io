    GMail Spam Filter using ML

    I created a spam filter using the Naive Bayes algorithm, trained on labeled email data. After evaluating its performance, I deployed the filter to a Gmail account via Google Cloud, enabling real-time spam detection. The system was monitored and optimized to adapt to evolving spam tactics


    Encrypting an Image using a Hilbert Curves

    In this project, I used the Hilbert curve to transform an image into a text file, encoding the image's data in a sequential manner. Additionally, I also applied Discrete Wavelet Transform (DWT) to remove any embedded steganography, ensuring the resulting text file is free from hidden data


    Testing Different Models for Stock Prediction

    In this project, I evaluated several models to predict stock prices, including ARIMA for time series forecasting, LSTM networks for capturing long-term dependencies, Linear Regression for a straightforward approach, and Gradient Boosting for its ensemble learning capabilities. Each model was tested for accuracy and effectiveness in stock price prediction.
